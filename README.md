# 🐍 贪吃蛇游戏

一个用 Python + curses 编写的经典贪吃蛇游戏。

## 特性

- 🎮 经典贪吃蛇玩法
- ⌨️ 支持方向键和 WASD 控制
- 📊 实时分数显示
- ⚡ 随分数增加速度加快
- 🔄 游戏结束后可按 R 重新开始
- 🎨 终端图形界面

## 运行要求

- Python 3.6+
- 支持 curses 的终端（Linux/macOS/Windows with WSL）

## 安装与运行

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/snake-game.git
cd snake-game

# 运行游戏
python3 snake_game.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 控制方向 |
| W A S D | 控制方向（备选） |
| Q | 退出游戏 |
| R | 重新开始（游戏结束后） |

## 游戏规则

- 控制蛇移动，吃到食物（●）获得 10 分
- 蛇不能撞到墙壁或自己的身体
- 每获得 50 分，游戏速度会加快
